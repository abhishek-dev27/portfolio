## ✨ Features

- **🎨 Premium Dark UI** — Glassmorphism, smooth gradients, and custom color palette
- **⚡ Buttery Animations** — Scroll-triggered reveals, hover effects, and page transitions via Framer Motion
- **📱 Fully Responsive** — Optimized layout for mobile, tablet, and desktop
- **🧭 Sticky Navbar** — Scroll-aware with blur backdrop and mobile hamburger menu
- **🧩 Bento Grid Projects** — Dynamic project cards with GitHub links, tags, and bullet descriptions
- **🛠 Skills Section** — Categorized tech stack with animated skill badges
- **💼 Experience Timeline** — Professional journey with role details
- **📬 Contact Section** — Direct email CTA and social links
- **🌌 Particle Background** — Custom animated canvas background

## 🛠 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/abhishek-dev27/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be running at **http://localhost:5173**

### Build for Production

```bash
npm run build
npm run preview   # Preview the production build locally
```

## 🌐 Deployment

This project is deployed on **Vercel** with automatic builds on every push to `main`.

```bash
# Deploy manually via Vercel CLI
npx vercel --prod
```
